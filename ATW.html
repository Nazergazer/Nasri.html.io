About The Website

Our Goal

Our goal is to create a website that analysis YouTube data and be able to provide it to content creators of YouTube, viewers or users of YouTube and collaboration opportunity

A description of the data the website contains
The data on the website will contain data that is sourced from YouTube. It will include statistical data such as the number of views or searches of the topic, the like-to-dislike ratio, the number of videos on the topic, and the comment counts. Videos or links might be selected to show the user examples of types of videos that are in the topics, to allow more understanding of the topics that they choose. Visual graphs might be presented to allow better understanding for the users.
